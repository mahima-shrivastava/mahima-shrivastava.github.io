## Image Optimization Tips

1. **Format**: Use WebP for best compression, JPG for photos, PNG for graphics
2. **Size**: Optimize images to reduce file size
3. **Dimensions**: 
   - Case study images: 80x80px (will scale)
   - PULLED images: Square format recommended
   - Illustrations: SVG format preferred

## Tools for Optimization

- [TinyPNG](https://tinypng.com/) - Image compression
- [Squoosh](https://squoosh.app/) - Image optimization tool
- [ImageOptim](https://imageoptim.com/) - Batch optimization
